# Erstellung von Parametern

# **Erstellung von Parametern**

_Bei der Erstellung von Parametern soll eine einheitliche Vorgehensweise gelten. Folgendes "wording" wird festgelegt:
Abteilung / Sonderfall 1 / Sonderfall 2 / Parameter / Quant./Qual. / NWG / Kunde_

### **z.B.:**
- M Tupfer GKZ <5 Aldi Bake Off
- M Kratzschwamm Listerien quant <100 
- M mesophil anaerob GKZ quant <100 Lidl
- Zur besseren Übersichtlichkeit können folgende Abkürzungen verwendet werden:

### **Abkürzungsliste**

- Gesamtkeimzahl: GKZ
- Milchsäurebakterien: Lakto
- Enterobacteriaceae: Enteros
- Salmonella: Salmo
- Listeria spp: Listerien
- Listeria monocytogenes: L.mono
- Escherichia coli: E.coli
- Shigatoxin-bildende E. coli: STEC
- Verotoxinbildende E. coli: STEC
- Enterohämorhagische E. coli: EHEC
- Enteroaggregative E. coli: EAEC
