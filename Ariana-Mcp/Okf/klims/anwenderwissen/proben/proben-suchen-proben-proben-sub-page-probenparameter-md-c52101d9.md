# Probenparameter

# **Probenparameter**
Parameter:
- Bezeichnung: exakter Name des Basisparameters oder mit Platzhalter *
- Bezeichnung: Name mit % wie folgt umschreiben: z.B "C Dekor in % durch "C Dekor in %%" ersetzen (Das wird im Hintergrund für die Datenbankabfrage als % interpretieret)
- Inhalt:  ~* mit Inhalt * sucht alle Proben ergebnisunabhängig, in denen der Parameter vorhanden ist.
