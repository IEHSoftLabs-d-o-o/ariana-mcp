# Auftragsplanung

# **Auftragsplanung**
## **Auftretende Probleme**
**Backsystem war falsch hinterlegt:**
- Probe in Klims stornieren
- In der zugehörigen Filiale das System korrigieren
- Neuen Planauftrag für das neue System erzeugen und aktivieren
- Übertragen der Daten des Blankozettels in die offizielle Checkliste

**Ein System konnte nicht bearbeitet werden:**
- Im Planauftrag das System stornieren (nicht den Gesamtauftrag)
- In der Filiale den abweichenden **Probenahmetermin für das System** setzen
- Neuen Planauftrag für das neue System erzeugen und aktivieren
